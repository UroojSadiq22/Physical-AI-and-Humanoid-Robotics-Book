This document provides a quick overview of how to get started with the textbook.

## Prerequisites

-   A computer running Ubuntu 22.04
-   ROS 2 Humble Hawksbill installed
-   Gazebo installed
-   Unity Hub installed
-   NVIDIA Isaac Sim installed
-   Docusaurus installed

## Getting Started

1.  **Clone the repository**: `git clone <repository-url>`
2.  **Navigate to the `docs` directory**: `cd docs`
3.  **Install dependencies**: `npm install`
4.  **Start the development server**: `npm start`
5.  **Open your browser** to `http://localhost:3000` to see the textbook.
